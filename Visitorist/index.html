THE VISITORIST
A City Guide Map
The VISITORIST (Visitor/Tourist) provides a simple interface for someone looking to tour the city. Simply scroll through the page. There are no map navigation tools. Data is hosted on ArcGIS Online. Click on place names on the right for detailed information.

This website has been built by Aamir Suleman using an open source library called Skrollr. Aamir is a JavaScript Developer at Blue Raster in Arlington, VA.
